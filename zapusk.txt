как запускать входную точку в приложение (main class) ИЛИ ПРОСТО ПРИЛОЖЕНИЕ

___________________________________________________

ВАРИАНТ 1:

через команду ``` ./gradlew run ```

ВАРИАНТ 2:

через IDEA но нужно будет добавить доп параметры запуска HelloApplicaton а именно

```  --module-path /usr/lib/jvm/openjfx/lib --add-modules  javafx.controls,javafx.fxml ```

в IDE по другому никак не запустить fx приложение (нужно указывать параметры VM)
