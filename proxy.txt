Как отключать прокси на пк 

___________________________________________________

ВАРИАНТ 1:

посмотреть что находится в скрипте 

``` cat proxy.sh ```

ЕСЛИ ТАМ ЭКСПОРТЫ ТО ПРОСТО ВВОДИМ КОМАНДЫ

``` unset http_proxy
    unset https_proxy
    unset ftp_proxy
    unset no_proxy
 ```


ПОСЛЕ ЧЕГО ПОСМОТРЕТЬ НЕТ ЛИ ПРОКСИ В АВТОЗАГРУЗКЕ

``` nano ~/.bashrc ```

ЕСЛИ БУДЕТ СТРОЧКА

``` source  ~/proxy.sh ``` ТО НУЖНО ЕЁ ЗАКОММЕНТИРОВАТЬ ИЛИ УДАЛИТЬ

ПОСЛЕ ЧЕГО ОБНОВИТЬ КОНФИГУРАЦИЮ

``` source ~/.bashrc ```


ЧТОБЫ ЗАПУСТИТЬ ОБРАТНО НУЖНО ВЫПОЛНИТЬ КОМАНДУ: "source ~/proxy.sh "
